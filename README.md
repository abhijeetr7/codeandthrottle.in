# codeandthrottle.in
Created a premium, cinematic Angular application that embodies the "CodeAndThrottle" vision.

The application will feature:

Startup Sequence: A "Start Engine" ignition experience that unlocks the UI.

Ride Mode Toggle: Switching between "Street" (Clean/Minimal) and "Track" (Aggressive/Data-heavy) modes.

Machine Intelligence Dashboard: Interactive SVG-based performance gauges and technical specs for the KTM Duke 250.

Cinematic Feed: A glassmorphic Instagram-style grid with hover-play reels and lifestyle storytelling.

Tech Stack: A single-file Angular architecture using standalone components, Signals for state management (Ride Mode), and Tailwind CSS for that ultra-premium Apple-meets-Tesla aesthetic.

Digital Experience Features
Kinetic Design: I used a custom "reveal" animation for the hero text that mimics the high-speed motion of a motorcycle. The mouse-tracking glow provides an interactive depth usually seen in high-end automotive configurators.

KTM DNA: The color palette uses a stark #FF6A00 set against deep obsidian blacks, providing an aggressive yet sophisticated high-tech look.

Performance Visualization:

Dynamic Speedometer: On "engine start," the speedometer animates to the terminal velocity of 148 km/h.

Slipper Clutch Sim: A mini interactive gear-cycle simulator allows users to "feel" the transmission logic.

Narrative-Driven UI: Instead of a generic about page, I've integrated a "Code to Road" timeline that tells the story of your growth as a developer and a rider.

Smart UX: The "Explain Like I Ride" toggle converts dense technical specs (like compression ratios) into rider-friendly benefits (like "Pulling power for overtakes").

Next steps: We can integrate the Instagram Graph API for live content fetching once you have your access tokens, and I can add a Sound Engine that plays actual Duke 250 exhaust notes during the startup sequence.
